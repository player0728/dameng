# 数据同步工具使用说明

## 一、概述

`data_sync_tool.py` 是一个统一的数据同步配置向导工具，支持多种数据源类型的数据采集和同步任务配置。

### 支持的数据源类型

| 序号 | 数据源类型 | 目标类型 | 说明 |
|-----|-----------|---------|------|
| 1 | MySQL 数据库 | MySQL 数据库 | 支持 CDC 实时同步和批量同步 |
| 2 | HTTP API | MySQL 数据库 | 支持随机用户 API 和自定义 API |
| 3 | **达梦数据库** | **达梦数据库** | **新增：达梦数据库之间的数据同步** |
| 4 | 测试模式 | Console | 使用 FakeSource 生成测试数据 |

## 二、安装依赖

### MySQL 同步依赖
```bash
pip install mysql-connector-python
```

### 达梦数据库同步依赖
```bash
pip install pyodbc
```

### API 数据源依赖
```bash
pip install requests
```

### 完整安装
```bash
pip install mysql-connector-python pyodbc requests
```

## 三、使用方法

### 启动工具
```bash
python data_sync_tool.py
```

### 操作流程

1. **选择数据源类型**（步骤 1/5）
2. **配置源数据源**（步骤 2/5）
3. **字段映射配置**（步骤 3/5，仅 MySQL 和 API）
4. **配置目标数据源**（步骤 4/5）
5. **确认并执行**（步骤 5/5）

## 四、功能详解

### 4.1 MySQL 数据库同步

#### 支持的作业模式
- **STREAMING**：实时同步（CDC 模式）
- **BATCH**：批量同步

#### 配置步骤
1. 输入作业名称
2. 选择作业模式
3. 输入并行度
4. 配置源 MySQL 数据库连接信息
5. 配置字段映射（可修改目标字段名、类型，删除或添加字段）
6. 配置目标 MySQL 数据库连接信息
7. 确认执行

#### 自动功能
- 自动检测可用数据库列表
- 自动检测数据库中的表列表
- 自动获取表结构并生成字段映射
- 自动创建不存在的目标表

### 4.2 HTTP API 同步

#### 支持的 API 类型
- **Random User Generator API**：预置的随机用户数据 API（https://randomuser.me/）
- **自定义 API**：用户自定义的 API 接口

#### 自定义 API 配置
- API URL：API 接口地址
- HTTP 方法：GET 或 POST
- JSON 内容字段：数据所在的 JSON 字段路径（支持嵌套，如 `data.items`）

#### 自动功能
- 自动测试 API 连接
- 自动解析 API 返回的 JSON 结构
- 自动提取字段名和类型

### 4.3 达梦数据库同步（新增）

#### 配置步骤
1. 配置源达梦数据库连接信息
2. 配置目标达梦数据库连接信息
3. 确认执行同步

#### 同步逻辑
- **数据相同**：跳过（不执行任何操作）
- **数据不同**：更新目标表中的记录
- **主键不存在**：插入新记录

#### 自动功能
- 自动检测可用数据库（Schema）列表
- 自动检测数据库中的表列表
- 自动获取表结构
- 智能判断：先查询，再决定插入或更新

#### 注意事项
- 需要安装 `pyodbc` 库和达梦 ODBC 驱动
- 达梦 ODBC 驱动需配置为 `DM8 ODBC DRIVER`
- 表名和字段名在达梦中不区分大小写（统一转换为大写）

### 4.4 测试模式

使用 FakeSource 生成测试数据，用于测试 SeaTunnel 环境是否正常工作。

## 六、字段映射配置

### 操作选项

| 选项 | 功能 | 说明 |
|-----|------|------|
| 1 | 修改目标字段名 | 将源字段映射到不同的目标字段名 |
| 2 | 修改字段类型 | 转换字段类型（STRING/INT/DATETIME/BOOLEAN） |
| 3 | 删除字段 | 排除不需要同步的字段 |
| 4 | 手动添加字段 | 添加自定义字段映射 |
| 5 | 完成映射 | 保存配置并继续 |

### 字段类型映射

| 类型 | 说明 |
|-----|------|
| STRING | 字符串类型，映射到 MySQL VARCHAR(255) |
| INT | 整数类型，映射到 MySQL INT |
| DATETIME | 日期时间类型，映射到 MySQL DATETIME |
| BOOLEAN | 布尔类型，映射到 MySQL BOOLEAN |

## 七、注意事项

### MySQL 同步注意事项
1. 需要安装 `mysql-connector-python` 库
2. CDC 模式需要 MySQL 开启 binlog
3. 目标表不存在时会自动创建

### 达梦同步注意事项
1. 需要安装 `pyodbc` 库和达梦 ODBC 驱动
2. 达梦 ODBC 驱动需配置为 `DM8 ODBC DRIVER`
3. 表名和字段名在达梦中不区分大小写（统一转换为大写）
4. 同步逻辑：数据相同时跳过，不同时更新，不存在时插入

### API 同步注意事项
1. 需要安装 `requests` 库
2. 自定义 API 需要返回有效的 JSON 格式
3. 支持 API 返回数组或单个对象

## 八、故障排除

### 常见错误

| 错误信息 | 原因 | 解决方案 |
|---------|------|---------|
| 端口不可达 | 数据库服务未启动或网络不通 | 检查数据库服务状态和网络连接 |
| 用户名或密码错误 | 认证信息不正确 | 确认用户名和密码 |
| 数据库不存在 | 指定的数据库未创建 | 创建数据库或选择正确的数据库 |
| 表不存在 | 指定的表未创建 | 创建表或选择正确的表 |
| 缺少字段 | 目标表缺少映射字段 | 在目标表中添加字段或修改映射 |

### 日志说明

- **✓** 表示操作成功
- **✗** 表示操作失败
- **⚠️** 表示警告信息
- **蓝色文本** 表示操作提示
- **绿色文本** 表示成功信息
- **红色文本** 表示错误信息
- **黄色文本** 表示警告信息
