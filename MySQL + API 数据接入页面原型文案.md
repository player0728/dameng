# 数据接入配置页面

---

## 第一步：选择数据源类型

> 页面顶部，用 Tab 或 Radio 切换

| 选项 | 说明 |
|------|------|
| 🗄️ MySQL | 从 MySQL 数据库中读取数据 |
| 🌐 API 接口 | 从 HTTP 接口中获取数据 |

---

## 第二步A：MySQL 连接配置

> 选择 MySQL 后，展示以下表单

### 基本连接信息

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 数据源名称 | 输入框 | ✅ | 给这个数据源起个名字 |
| 主机地址 | 输入框 | ✅ | 如：192.168.1.1 |
| 端口 | 数字输入框 | ✅ | 默认 3306 |
| 数据库名 | 输入框 | ✅ | 如：my_database |
| 用户名 | 输入框 | ✅ | 如：root |
| 密码 | 密码框 | ✅ | 密文显示 |

### 操作按钮
- 【测试连接】：点击后显示"连接成功 ✅"或"连接失败 ❌ + 错误原因"

---

### 数据表选择

> 连接成功后展示以下内容

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 选择数据库 | 下拉选择 | ✅ | 拉取数据库列表供选择 |
| 选择数据表 | 下拉选择 | ✅ | 拉取该库下的表列表 |
| 选择字段 | 多选框 | ✅ | 展示表字段，默认全选 |

---

### SQL 配置（可选）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 自定义 SQL | 文本域 | ❌ | 填写后覆盖上方表/字段选择 |
| WHERE 条件 | 输入框 | ❌ | 如：status = 1 |
| 数据条数限制 | 数字输入框 | ❌ | 如：1000，默认不限制 |

---

### 增量同步配置（可选）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 是否开启增量 | 开关 | ❌ | 默认关闭 |
| 增量字段 | 下拉选择 | 开启后必填 | 选择 id 或 update_time 等 |
| 起始值 | 输入框 | ❌ | 如：0 或 2024-01-01 |

---

## 第二步B：API 接口配置

> 选择 API 后，展示以下表单

### 基本信息

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 数据源名称 | 输入框 | ✅ | 给这个接口起个名字 |
| 请求地址 | 输入框 | ✅ | 完整 URL，如：https://api.xxx.com/data |
| 请求方式 | 下拉选择 | ✅ | GET / POST |
| Content-Type | 下拉选择 | ❌ | application/json（默认）|

---

### 请求参数配置

> 以下三项均为动态表单，支持"+ 添加一行"

#### Query 参数（Params）
| Key | Value |
|-----|-------|
| 输入框 | 输入框 |

#### 请求头（Headers）
| Key | Value |
|-----|-------|
| 输入框 | 输入框 |

#### 请求体（Body，POST 时显示）
- 文本域，填写 JSON 格式内容
- 如：`{"page": 1, "size": 100}`

---

### 认证方式（可选）

| 选项 | 说明 |
|------|------|
| 无认证 | 默认 |
| Bearer Token | 填写 Token 值 |
| Basic Auth | 填写用户名 + 密码 |
| API Key | 填写 Key 名称 + 值 |

---

### 返回结果配置

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 数据节点路径 | 输入框 | ❌ | 如：data.list，用于提取数组 |
| 返回格式 | 下拉选择 | ✅ | JSON（默认）/ Text |

### 操作按钮
- 【发送请求 / 测试接口】：点击后展示接口返回的原始 JSON 内容

---

## 第三步：字段映射（通用）

> MySQL 和 API 均有此步骤

| 源字段名 | 目标字段名 | 字段类型 | 操作 |
|----------|------------|----------|------|
| id | id | INT | 删除 |
| name | user_name | STRING | 删除 |
| create_time | created_at | DATETIME | 删除 |

- 支持修改目标字段名
- 支持修改字段类型（STRING / INT / DATETIME / BOOLEAN）
- 支持删除不需要的字段
- 【+ 手动添加字段】按钮

---

## 第四步：任务调度配置

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 任务名称 | 输入框 | ✅ | 如：用户数据同步任务 |
| 执行方式 | Radio | ✅ | 手动执行 / 定时执行 |
| Cron 表达式 | 输入框 | 定时时必填 | 如：0 0 * * *（每天零点）|
| 失败重试次数 | 数字输入框 | ❌ | 默认 3 次 |

---

## 第五步：数据预览

> 点击【预览数据】按钮，展示前 10 条数据

| 字段1 | 字段2 | 字段3 | ... |
|-------|-------|-------|-----|
| 数据行 | 数据行 | 数据行 | ... |

- 展示条数：默认 10 条
- 提示文字：「以下为预览数据，仅供参考，不影响实际同步」

---

## 底部操作按钮

| 按钮 | 说明 |
|------|------|
| 【上一步】 | 返回上一步 |
| 【保存草稿】 | 保存当前配置，不执行 |
| 【立即执行】 | 保存并立即运行任务 |
| 【取消】 | 放弃当前配置 |

---

## 附：页面整体步骤导航（顶部 Steps 组件）

```
① 选择类型  →  ② 连接配置  →  ③ 字段映射  →  ④ 调度配置  →  ⑤ 预览确认
```
